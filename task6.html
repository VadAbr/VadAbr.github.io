Дан код деструктуризации в существующую переменную, который выдаёт синтаксическую ошибку:

let a; { a } = { a: 5 }; console.log(a);

Вам нужно исправить его, добавив к нему два символа. Это единственное исправление, которое нужно сделать. После исправления вывод в консоль даст значение 5. 

    let a; ({ a } = { a: 5 }); console.log(a);
