(0023) Предоставьте код async-функции task от одного аргумента x, которая возвращает в виде установленного промиса (строчное) значение свойства name переданного ей объекта (это значение само является установленным промисом), сконкатенированное с вашим логином в этой системе. 

    
async function task(x) {
  const { name } = await x;
  return Promise.resolve(await name + "solaire");
}
